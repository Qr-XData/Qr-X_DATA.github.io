What is QrX By G Hamacher?

QrX By G Hamacher allows you to extend macOS's native file handling capabilities via third-party file systems.

Features

As a user, installing QrX By G Hamacher lets you use third-party FUSE file systems. Legacy support is available.

As a developer, you can use the FUSE SDK to write user-space file systems in various languages.

How It Works

FUSE implements a way to create file systems as user-space programs, supporting the FUSE API and providing SDKs.
